# JavaScript Fundamentals: The indexOf() Method

## 1. What is a Substring?
* **Definition:** A sequence of characters that exists within a larger string.
* **Example:** In the string `"hello world"`, both `"hello"` and `"world"` are substrings.

## 2. The indexOf() Method
* **Purpose:** To find the starting position (index) of a specific substring.
* **Syntax:** `string.indexOf(substring, startPosition)`
* **Arguments:**
    1. **Substring (Required):** The text you are looking for.
    2. **Start Position (Optional):** The index where the search should begin. Defaults to `0`.

## 3. Understanding the Return Values
* **Found:** Returns the **index number** of the first occurrence of the substring.
* **Not Found:** Returns **-1**. This is a standard way in JavaScript to indicate a search was unsuccessful.

## 4. Key Behaviors

### A. Case Sensitivity
* The method is strictly case-sensitive. Searching for `"F"` in `"freeCodeCamp"` will return `-1` because the string only contains a lowercase `"f"`.

### B. Specifying a Starting Point
* By providing a second argument, you can skip the beginning of a string to find subsequent occurrences.
* **Example:**
  ```javascript
  let str = "Code, Code, Code";
  console.log(str.indexOf("Code", 5)); // Returns 6 (skips the first "Code")
