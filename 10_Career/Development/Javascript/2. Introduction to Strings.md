# JavaScript Fundamentals: Introduction to Strings

## 1. What is a String?
* **Definition:** A string is a sequence of characters used to represent text data.
* **Type:** It is one of the **primitive** data types in JavaScript, alongside numbers, Booleans, null, and undefined.

## 2. Syntax & Creation
Strings are created by wrapping text in quotes. You must be **consistent**—a string must start and end with the same type of quote.

* **Single Quotes:** `'This is a string'`
* **Double Quotes:** `"This is also a string"`
* **Invalid Syntax:** `"Do not mix quotes' ` (This will throw a **Syntax Error**).

## 3. String Immutability
* **Definition:** In programming, immutability means that once a value is created, it **cannot be changed**.
* **Direct Modification:** You cannot change individual characters within a string (e.g., you cannot "reach in" and change the first letter of an existing string).
* **Updating a Variable:** To "change" a string stored in a variable, you must assign a **completely new string** to that variable.
    * *Example:*
        ```javascript
        let developer = "Jessica"; 
        // You cannot change "J" to "M" directly.
        developer = "Quincy"; // You replace the old string with a new one.
        ```

## 4. Key Takeaways
* Strings represent text.
* Consistency in quote marks is mandatory.
* Strings are **immutable**; they are replaced, not edited in place.
