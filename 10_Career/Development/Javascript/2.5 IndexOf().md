	# Detailed Study Notes: Finding the Position of a Substring in a String

## 1. Core Concept: Substrings
* **Definition:** A substring is a sequence of characters that appears within a larger string.
* **Example:** In the string `"hello world"`, both `"hello"` and `"world"` are considered substrings.

---

## 2. The `indexOf()` Method
The `indexOf()` method is the primary tool in JavaScript for locating a substring within a larger string.

### Key Behaviors:
* **Successful Search:** Returns the **index (position)** of the first occurrence of the specified substring.
* **Unsuccessful Search:** Returns **`-1`** if the substring is not found.
* **Case Sensitivity:** The method is strictly case-sensitive (e.g., `"F"` is not the same as `"f"`).

---

## 3. Syntax and Arguments
The method accepts two parameters (arguments):
1.  **First Argument (Required):** The substring you want to search for.
2.  **Second Argument (Optional):** The starting position for the search.
    * If omitted, the search begins at the start of the string (index 0).
    * If provided, the method skips all characters before that index.

---

## 4. Practical Examples

### A. Basic Usage
Locating the word "awesome":
```javascript
let sentence = "JavaScript is awesome!";
let position = sentence.indexOf("awesome!");
console.log(position); // Output: 14
```
### B. Handling Missing Substrings

Searching for a word that does not exist:
```js
let sentence = "JavaScript is awesome!";
let position = sentence.indexOf("fantastic");
console.log(position); // Output: -1
```

### C. Using the Starting Position Argument

Finding a second occurrence by skipping the first:
```js
let sentence = "JavaScript is awesome, and JavaScript is powerful!";
let position = sentence.indexOf("JavaScript", 10);
console.log(position); // Output: 27
```
Note: By starting at index 10, the search ignores the "JavaScript" at index 0 and finds the one at index 27.
D. Case Sensitivity Demonstration:
console.log("freeCodeCamp".indexOf("F")); // Output: -1_Note: Returns -1 because "freeCodeCamp" contains a lowercase "f", not an uppercase "F"._

## 5. Assessment Summary (Quiz Logic)

- **Question:** What does `indexOf` return if not found?
    
    - **Answer:** `-1`.
        
- **Question:** How do you search starting at a specific position?
    
    - **Answer:** By using the **second argument**.
        
- **Question:** What will `str.indexOf("Javascript")` return for `"I am learning JavaScript."`?
    
    - **Answer:** `-1` (Due to the lowercase 's' in the search term vs uppercase 'S' in the string).
