# JavaScript Fundamentals: String Concatenation

## 1. What is Concatenation?
* **Definition:** The process of joining or combining multiple pieces of text (strings) into one.
* **Core Concept:** Because strings are **immutable**, concatenation does not modify the original strings. Instead, it creates a brand-new string.

## 2. The Addition Operator (`+`)
The simplest and most common way to join strings.
* **Usage:** Combine string literals or variables.
* **Important Note:** You must manually manage spacing. Forgetting a space between variables will result in the words being "glued" together.
* **Example:**
    ```javascript
    let firstName = "John";
    let lastName = "Doe";
    let fullName = firstName + " " + lastName; // Result: "John Doe"
    ```

## 3. The Compound Assignment Operator (`+=`)
Used when you want to build or append text to an existing string variable over time.
* **Usage:** `variable += "new text"`
* **Example:**
    ```javascript
    let greeting = "Hello";
    greeting += ", John!"; // Result: "Hello, John!"
    ```

## 4. The `.concat()` Method
A built-in method specifically designed for joining multiple strings.
* **Difference between Function and Method:**
    * **Function:** A reusable block of code that performs a task.
    * **Method:** A function associated with an object (like a string) that operates on that object's data.
* **Usage:** `string1.concat(string2, string3, ...)`
* **Example:**
    ```javascript
    let str1 = "Hello";
    let str2 = "World";
    let result = str1.concat(" ", str2); // Result: "Hello World"
    ```

## 5. Quick Comparison
| Tool | Best Use Case |
| :--- | :--- |
| `+` | Simple combinations of a few strings/variables. |
| `+=` | Appending content or building a string step-by-step. |
| `.concat()` | Concatenating many strings together at once. |
