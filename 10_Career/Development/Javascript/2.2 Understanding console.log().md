# JavaScript Fundamentals: Understanding console.log()

## 1. What is console.log()?
* **Definition:** A built-in JavaScript method used to display (or "log") messages, variables, or data to the browser's console.
* **Primary Purpose:** It is the go-to tool for **debugging** and **inspecting** code. It allows developers to "peek" inside their program while it is running.

## 2. Basic Syntax
To use it, you call the method and pass the information you want to see inside the parentheses.
* **Logging Strings:** `console.log("Hello, world!");`
* **Logging Variables:** ```javascript
  let num = 5;
  console.log(num); // Output: 5
