# JavaScript Data Types Reference

| Category      | Type      | Description                        |
| :------------ | :-------- | :--------------------------------- |
| **Primitive** | String    | Textual data in quotes.            |
| **Primitive** | Number    | Integers and decimals.             |
| **Primitive** | Boolean   | true or false.                     |
| **Primitive** | Undefined | Declared but not assigned.         |
| **Primitive** | Null      | Intentional empty value.           |
| **Primitive** | BigInt    | Very large integers (ends in 'n'). |
| **Primitive** | Symbol    | Unique identifiers.                |
| **Complex**   | Object    | Key-value pairs / Collections.     |

*Remember: Primitives are passed by value, while Objects are passed by reference.*
