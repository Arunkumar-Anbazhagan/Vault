1. can be installed using the npm 
2. npm means node package manager which is largest software registry and 
	1. All **npm** packages are defined in files called **package.json**.
	2. The content of package.json must be written in **JSON**.
	3. At least two fields must be present in the definition file: **name** and **version**.
		```json
		{  
"name" : "foo",  
"version" : "1.2.3",  
"description" : "A package for fooing things",  
"main" : "foo.js",  
"keywords" : ["foo", "fool", "foolish"],  
"author" : "John Doe",  
"licence" : "ISC"  
}
		```
