## Basic usage pattern (typical workflow)

- Find an app: `winget search sourcetree`
    
- Install it: `winget install --id Atlassian.Sourcetree`
    
- Check for updates later: `winget upgrade` then `winget upgrade --all`
    
- Move setup to a new machine: `winget export -o packages.json` on old PC, `winget import -i packages.json` on new PC.
