## Basic usage pattern (typical workflow)

- Find an app: `winget search sourcetree`
    
- Install it: `winget install --id Atlassian.Sourcetree`
    
- Check for updates later: `winget upgrade` then `winget upgrade --all`
    
- Move setup to a new machine: `winget export -o packages.json` on old PC, `winget import -i packages.json` on new PC.
## Usage and syntax
| Flag / Option                   | Syntax example                                                 | Usage / Meaning                                                                                                                                     |
| ------------------------------- | -------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------- |
| `-h`, `--help`, `-?`            | `winget install --help`                                        | Show help for the selected command.learn.microsoft+1                                                                                                |
| `-q`, `--query`                 | `winget uninstall -q powertoys`                                | Search string to identify a package (name, ID, etc.).[learn.microsoft](https://learn.microsoft.com/fi-fi/windows/package-manager/winget/uninstall)  |
| `-e`, `--exact`                 | `winget install powertoys -e`                                  | Match the query exactly (no partial matches, case‑sensitive).learn.microsoft+1                                                                      |
| `--id`                          | `winget install --id Atlassian.Sourcetree`                     | Filter by exact package ID.learn.microsoft+1                                                                                                        |
| `--name`                        | `winget install --name "Google Chrome"`                        | Filter by package name.learn.microsoft+1                                                                                                            |
| `--moniker`                     | `winget install --moniker chrome`                              | Filter by moniker (short alias).learn.microsoft+1                                                                                                   |
| `-v`, `--version`               | `winget install powertoys --version 0.15.2`                    | Use a specific version of a package.learn.microsoft+1                                                                                               |
| `-s`, `--source`                | `winget uninstall powertoys --source winget`                   | Limit to a specific source (like `winget`, `msstore`).learn.microsoft+1                                                                             |
| `--scope`                       | `winget install <id> --scope machine`                          | Choose install/uninstall scope: `user` or `machine`.learn.microsoft+1                                                                               |
| `-l`, `--location`              | `winget install <id> --location "D:\Apps"`                     | Custom install directory, if the installer supports it.[learn.microsoft](https://learn.microsoft.com/en-us/windows/package-manager/winget/install)  |
| `-o`, `--log`                   | `winget install <id> --log C:\temp\winget.log`                 | Write logs to the specified file path.learn.microsoft+1                                                                                             |
| `--silent`, `-h` (uninstall)    | `winget uninstall <id> --silent`                               | Run installer/uninstaller in silent mode, no UI.learn.microsoft+1                                                                                   |
| `-i`, `--interactive`           | `winget uninstall <id> --interactive`                          | Force interactive UI for the uninstaller.[learn.microsoft](https://learn.microsoft.com/fi-fi/windows/package-manager/winget/uninstall)              |
| `--force`                       | `winget install <id> --force`                                  | Continue even when non‑security issues occur.learn.microsoft+1                                                                                      |
| `--ignore-security-hash`        | `winget install <id> --ignore-security-hash`                   | Skip hash check failures (not recommended).[learn.microsoft](https://learn.microsoft.com/en-us/windows/package-manager/winget/install)              |
| `--allow-reboot`                | `winget install <id> --allow-reboot`                           | Allow the installer to reboot the system if needed.[learn.microsoft](https://learn.microsoft.com/en-us/windows/package-manager/winget/install)      |
| `--custom`                      | `winget install <id> --custom "/D=D:\Apps\App"`                | Extra arguments forwarded to the installer.learn.microsoft+1                                                                                        |
| `--override`                    | `winget install <id> --override "/S /D=D:\Apps\App"`           | Raw command‑line string passed directly to the installer.learn.microsoft+1                                                                          |
| `--uninstall-previous`          | `winget install <id> --uninstall-previous`                     | Uninstall older version during an upgrade‑style install.[learn.microsoft](https://learn.microsoft.com/en-us/windows/package-manager/winget/install) |
| `--accept-package-agreements`   | `winget install <id> --accept-package-agreements`              | Auto‑accept package license/terms prompts.[learn.microsoft](https://learn.microsoft.com/en-us/windows/package-manager/winget/install)               |
| `--accept-source-agreements`    | `winget source update --accept-source-agreements`              | Auto‑accept source/repository agreements.learn.microsoft+1                                                                                          |
| `--authentication-mode`         | `winget install <id> --authentication-mode silentPreferred`    | Choose auth window behavior: `silent`, `silentPreferred`, `interactive`.learn.microsoft+1                                                           |
| `--authentication-account`      | `winget install <id> --authentication-account user@domain.com` | Account to use for authentication.learn.microsoft+1                                                                                                 |
| `--purge`                       | `winget uninstall <id> --purge`                                | Delete all package files (portable packages).[learn.microsoft](https://learn.microsoft.com/fi-fi/windows/package-manager/winget/uninstall)          |
| `--preserve`                    | `winget uninstall <id> --preserve`                             | Keep files created by a portable package.[learn.microsoft](https://learn.microsoft.com/fi-fi/windows/package-manager/winget/uninstall)              |
| `--wait`                        | `winget uninstall <id> --wait`                                 | Wait for key press before exiting.[learn.microsoft](https://learn.microsoft.com/fi-fi/windows/package-manager/winget/uninstall)                     |
| `--logs`, `--open-logs`         | `winget list --open-logs`                                      | Open the default logs directory.[learn.microsoft](https://learn.microsoft.com/fi-fi/windows/package-manager/winget/uninstall)                       |
| `--verbose`, `--verbose-logs`   | `winget install <id> --verbose-logs`                           | Enable detailed logging even if disabled in settings.learn.microsoft+1                                                                              |
| `--nowarn`, `--ignore-warnings` | `winget install <id> --ignore-warnings`                        | Suppress warning messages.[learn.microsoft](https://learn.microsoft.com/fi-fi/windows/package-manager/winget/uninstall)                             |
| `--disable-interactivity`       | `winget install <id> --disable-interactivity`                  | Try to disable interactive prompts.learn.microsoft+1                                                                                                |
| `--proxy`                       | `winget install <id> --proxy http://proxy:8080`                | Use a specific proxy for this command run.[learn.microsoft](https://learn.microsoft.com/fi-fi/windows/package-manager/winget/uninstall)             |
| `--no-proxy`                    | `winget install <id> --no-proxy`                               | Do not use any proxy for this command run.[learn.microsoft](https://learn.microsoft.com/fi-fi/windows/package-manager/winget/uninstall)             |
