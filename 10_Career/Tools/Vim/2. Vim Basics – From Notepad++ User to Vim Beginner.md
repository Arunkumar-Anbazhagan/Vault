# Vim Basics – From Notepad++ User to Vim Beginner

## Mental model: modes
- **Normal mode** (default): move around and run commands.
  - Press `Esc` to return here from any other mode.
- **Insert mode**: type text like in Notepad++.
  - Enter with `i` (insert before cursor), `a` (after cursor), or `o` (new line below).
- **Command-line mode**: run colon commands like save/quit.
  - From Normal mode, press `:` then type a command (e.g., `w`, `q`, `wq`). [web:2][web:31]

---

## Open, edit, save, quit
- Open file: `vim filename`
- Start typing:
  - From Normal → `i` to enter Insert mode.
- Save:
  - `Esc` → `:w` → Enter
- Quit:
  - `Esc` → `:q` → Enter
- Save and quit:
  - `Esc` → `:wq` → Enter
- Quit without saving:
  - `Esc` → `:q!` → Enter [web:2][web:27]

---

## Basic movement (Normal mode)
- Arrow keys work (ok for learning).
- Home-row movement:
  - `h` = left
  - `j` = down
  - `k` = up
  - `l` = right
- Line navigation:
  - `0` = start of line
  - `$` = end of line
- Bigger jumps:
  - `w` = next word
  - `b` = previous word
  - `gg` = start of file
  - `G` = end of file [web:4][web:26]

---

## Editing: delete, undo, redo (Normal mode)
- Delete character under cursor: `x`
- Delete current line: `dd`
- Undo: `u`
- Redo: `Ctrl+r` [web:2][web:27]

---

## Copy & paste (yank & put)
- Copy current line: `yy`
- Paste below current line: `p`
- Visual selection:
  - `v` → move cursor to select characters
  - Then:
    - `y` to copy selection
    - `p` to paste after cursor/line [web:4][web:30]

---

## Tiny practice routine
1. Open a file: `vim test.txt`.
2. `i` → type some text → `Esc`.
3. Move using `h j k l`, `0`, `$`, `gg`, `G`.
4. Use `dd`, `x`, `u`, `Ctrl+r`.
5. Use `yy` and `p`, and try `v` + movement + `y` + `p`. [web:2][web:28]
