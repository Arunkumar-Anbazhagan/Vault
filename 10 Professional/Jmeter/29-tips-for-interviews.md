# 29. Tips for Interviews   
## Section 30: Tips for Interviews   
### 1. Understand Core Concepts   
- Thread Groups, Samplers, Controllers   
- Correlation and Parameterization   
- Test execution workflow   
   
### 2. Hands-on Experience   
- Create sample test plans   
- Practice API testing   
- Learn CLI commands   
- Generate HTML reports   
   
### 3. Know Performance Metrics   
- Response time, Throughput, Error rate   
- Percentiles (90th, 95th, 99th)   
- Concurrent users vs Throughput   
   
### 4. Real-world Application   
- Be ready to discuss projects   
- Explain test scenarios   
- Describe challenges faced   
- Solution approaches   
   
### 5. Best Practices   
- When to use GUI vs CLI   
- How to optimize tests   
- Resource management   
- Realistic simulation   
   
### 6. Integration Knowledge   
- CI/CD integration   
- Jenkins/Maven/Gradle   
- Reporting and dashboards
