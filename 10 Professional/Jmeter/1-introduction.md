# 1. Introduction   
### What is Performance Testing?   
   
- Testing to determine system behavior under various load conditions   
- Measures speed, scalability, stability, and reliability   
- Types:   
    - **Load Testing**: Testing with expected user load   
    - **Stress Testing**: Testing beyond normal operational capacity   
    - **Spike Testing**: Sudden increase/decrease in load   
    - **Endurance Testing**: Testing over extended period   
    - **Volume Testing**: Testing with large amounts of data   
   
### Why JMeter?   
- Open-source and free   
- Platform-independent (Java-based)   
- Supports multiple protocols (HTTP, HTTPS, FTP, JDBC, SOAP, REST, JMS)   
- User-friendly GUI   
- Extensive community support and plugins   
- Can be integrated with CI/CD pipelines
